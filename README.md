# rs.c

rs.c is a RuneScape Classic client implemented in C.

## Building
To building rs.c, use CMake. 
```sh
$ git clone --recurse-submodules https://github.com/hikilaka/rs.c
$ cd rs.c && mkdir build && cd build
$ cmake ../ && make
```

## Dependencies

* libbz2-dev
* SDL2
